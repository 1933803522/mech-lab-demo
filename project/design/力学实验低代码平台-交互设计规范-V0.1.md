# 力学实验低代码平台交互设计规范 V0.1

> 本文档根据 `project/prd/力学实验低代码平台-PRD-V0.1.md` 中已确认的产品事实整理。
> 未经 PRD 明确确认的交互细节统一标记为“待确认”，不在本文档中推断补全。

## 1. 文档说明

### 1.1 文档目标

用于沉淀本期已确认页面的交互结构、通用操作规则、状态反馈规则和页面级交互要求，作为后续 UI 设计、原型设计和前端实现的协作依据。

### 1.2 信息来源

| 来源 | 说明 |
| :--- | :--- |
| PRD V0.1 | 已确认的页面范围、布局、流程、异常与边界状态 |

### 1.3 适用范围

| 范围 | 当前说明 |
| :--- | :--- |
| 登录页 | 账号密码登录 |
| 首页 | 后台管理类首页，左侧菜单栏，右侧显示对应内容 |
| 项目管理 | 增删改查；项目打开后的复杂内容暂不讨论 |
| 用户管理 | 增删改查 |
| 角色管理 | 增删改查 |
| 量纲管理 | 管理量纲和量纲下的单位 |
| 试样管理 | 管理试样类型和类型下的参数 |
| 切换用户 | 登录后提供切换用户入口，触发后返回登录页面 |

### 1.4 暂不纳入范围

| 内容 | 说明 |
| :--- | :--- |
| 项目打开后的复杂内容 | PRD 已明确暂不讨论 |
| 权限差异设计 | 当前管理员和普通用户在上述页面中暂时没有功能差异 |
| 接口细节 | 待补充 |
| 后端方案 | 待补充 |
| 视觉风格规范 | 颜色、字体、间距、圆角、图标风格等待确认 |

## 3. 通用操作规范

| 步骤 | 交互规则 |
| :--- | :--- |
| 1 | 用户从首页左侧菜单进入对应管理页面 |
| 2 | 用户在右侧内容区查看对应管理内容 |
| 3 | 用户在列表附近触发新增、编辑、删除、查看操作 |
